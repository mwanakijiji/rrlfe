You can install it with

pip install rrlfe

Docker and Singularity are options too.

Follow the instructions here <https://github.com/czwa/robospect.py> to install Robospect. Clone the code into the same parent directory in which the RRL package resides.

Change to the robospect.py/tmp/ directory and copy the file ll to it:

cp ../../rrlfe/ll .

This file tells Robospect which absorption lines to look for.

Users can file issues here.
